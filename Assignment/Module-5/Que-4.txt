4-What is MF Cod Rule of RDBMS Systems?

Codd's rule in DBMS also known as Codd's 12 rules/commandments is a set of thirteen 
rules (numbered 0 to 12) that define a database to be a correct Relational Database
 Management System (RDBMS). If a database follows Codd's 12 rules, it is called a 
 True relational database management system.
